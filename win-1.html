    Windows 1.0

    Windows 1.0

    Nakon što je Microsoft nekoliko godina eksperimentirao sa grafičkim sučeljima, odlučio je ovaj način komunikacije s korisnikom upotrijebiti u svom softveru. Tako je 20. studenog 1985. g. objavljen Windows 1.0. To je bio prvi Microsoftov pokušaj sa komercijalnom upotrebom grafičkog sučelja, a pokazao se ispravnom odlukom i korakom naprijed. Sam sustav je baziran na MS-DOS-u, imao je 16-bitnu ljusku koja je omogućila Windows-u ono što se danas podrazumijeva pri korištenju računala, a to je obavljanje više zadataka odjednom. Do tada su računala mogla pokrenuti samo jedan program, a onda ga zatvoriti pa pokrenuti drugi, Windows 1.0 je to promijenio, te je omogućio obavljanje više zadataka istovremeno. Samo sučelje se sastojalo od prozora, po čemu i nosi naziv (eng. window – prozor). Programi su se nalazili u tim prozorima, a na ekranu se moglo istovremeno koristiti nekoliko prozora odn. programa, bez preklapanja. Prozori su se mogli minimizirati u dok (eng. dock) koji se nalazio na dnu ekrana, a upravo taj dok je preteča današnjeg Start izbornika, koji će se pojaviti u kasnijim verzijama Windowsa. Za korištenje ovakvog sučelja miš je bio neophodan, pa je to izazvalo određeni kriticizam prema Windows 1.0 operacijskom sustavu. Unatoč tim kritikama Windows 1.0 je postao neizostavni softver svake tvrtke, te je utvrdio put sljedećim verzijama Windows-a koje imaju iza njega doći. Microsoft je ovu verziju Windows-a podržavao najduže od svih ostalih, punih 16 godina, pa je tako kraj podrške bio na samom kraju 2001. godine. Windows 1.0 će naslijediti Windows 2.0, 1987. godine.
