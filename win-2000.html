Windows 2000

Windows 2000 je prvi predstavnik Microsoft-ove Windows NT obitelji operacijskih sustava koju je Microsoft razvio u suradnji s IBM-om. Međutim, staze ovih tvrtki su se razišle zbog nesuglasica, pa je Microsoft odlučio prekinuti suradnju. Ipak, nastavili su razvoj Windows-a iz tog koda, te su uvelike povećani sigurnost, stabilnost i brzina sustava, pa se tako ova arhitektura počela koristiti i u komercijalnim operacijskim sustavima Windows, od kojih je prvi upravo Windows 2000. Sučelje sustava je ostalo jako slično prethodnim verzijama Windows-a, ali je jezgra sustava potpuno izmijenjena. Važno je napomenuti da Microsoft objavljuje Windows ME nakon objave Windows-a 2000, kako bi u slučaju tužbe od strane IBM-a zbog korištenja zajedničkog koda i dalje postojala verzija Windows-a za korisnike računala diljem svijeta. Windows 2000 je objavljen u više verzija: Professional, Server, Napredni server i Server baze podataka, ove serverske verzije su objavljene nešto kasnije, dok je Windows 2000 Professional objavljen 17. veljače 2000. godine. Windows 2000 sa sobom donosi novi sustav pohrane (NTFS 3.0), sustavnu enkripciju, promjenu jezika bez ponovne instalacije sustava, te brojne pomoćne alate za osobe s oštećenjima (povećalo, zaslonska tipkovnica, narator i sl.). Microsoft je smatrao Windows 2000 najsigurnijim operacijskim sustavom u povijesti operacijskih sustava, a Windows 2000 je svojom snažnom i stabilnom arhitekturom to potvrdio. Windows NT baza je uistinu bila korak naprijed za operacijske sustave Windows. Što se tiče osnovnih Windows-ovih programa, IE se popeo na verziju 6.0, a WMP na verziju 9 do kraja razvoja. Sa Windows-om 2000 dolazi i bolja podrška za više diskova koja omogućuje korištenje više fizičkih diskova kao jednog logičkog u svrhu boljih performansi ili bolje zaštite podataka, ovisno o potrebi. Od administrativnih alata Windows-a 2000 izdvajaju se novi preglednik događaja i konzola za povratak. Za Windows 2000 su izdana 4 servisna paketa, svaki godinu dana nakon prethodnog. Iako je planirana podrška za 64-bitni hardver, ona je došla tek s nasljednikom ovoga sustava. Windows 2000 je općenito napravio velik korak naprijed u svijetu operacijskih sustava tako što je donio sve alate za upravljanje farmama servera svakom kućnom korisniku Windows operacijskog sustava. Ovaj operacijski sustav je prvi Microsoftov sustav koji se objavio u više verzija, stvarajući tako tradiciju koja traje i danas. Opća podrška za sustav Windows 2000 je trajala do sredine 2005. godine.
