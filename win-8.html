Windows 8/8.1

Windows 8

Windows 8 je objavljen 26. listopada 2012. godine, a njegov razvoj je započeo nešto prije objavljivanja operacijskog sustava Windows 7. Windows 8 je najpoznatiji po svom Metro izgledu, koji zamjenjuje tradicionalni Start izbornik sa Start ekranom, koji programe i informacije prikazuje u obliku pločica, stvarajući tako sličan doživljaj onome koji se može pronaći na Microsoft-ovome mobilnom softveru Windows Phone. Windows 8 je prvi Windows operacijski sustav koji je imao mobilne verzije, jedna od njih je Windows Phone 8, koja nema istu arhitekturu kao i Windows 8 za PC ali ima slično sučelje i funkcije, te koristi istu trgovinu aplikacija (Windows Store); a druga verzija je Windows RT, koji je bio prvi pokušaj izvođenja Windows operacijskog sustava na ARM arhitekturi procesora koji se koriste u tabletima i mobilnim telefonima. Ova verzija nije mogla pokretati tradicionalne Windows programe, što su mnogi korisnici kritizirali. Windows 8 je bio izuzetno dobro prilagođen za upravljanje ekranima na dodir, što je moderno i suvremeno, ali zbog velike većine korisnika koji računalo koriste upravljajući mišem i tipkovnicom, Windows 8 je bio ozloglašen upravo zbog težeg upravljanja s tipkovnicom i mišem, te stavljanja dodirnog sučelja na prvo mjesto. Što se tiče razvojnih činjenica, Windows 8 donosi brojne nove mogućnosti i strukturalne promjene. Općenito se Windows 8 može smatrati sustavom Windows 7 na koji je dodana Metro ljuska sa tzv. živim pločicama (eng. Live Tiles), zato što je većina koda preuzeta iz operacijskog sustava Windows 7. Windows Aero tema je zamijenjena sa manje prozirnom temom bez teksturnih varijacija, Windows Explorer je preimenovan u File Explorer, dodano je novo stanje isključenog računala pod nazivom hibridno spavanje, koje je kombiniralo hibernaciju i potpuno isključivanje. Novi File Explorer koristi tzv. Ribbon umjesto tadašnjih alatnih traka, što se zadržalo sve do današnjih dana. Razvijen je nov zaslon zaključavanja, koji je omogućio nove načine otključavanja računala, kao npr. upotrebom PIN-a ili slike. Sa sustavom Windows 8, Microsoft započinje integraciju operacijskog sustava Windows sa svojom igraćom konzolom Xbox. Microsoft je sa operacijskim sustavom Windows 8 zapravo želio stvoriti Windows-ov ekosustav, u kome bi Windows mogao biti na uređajima raznih vrsta i oblika, uključujući računala, mobilne uređaje, tablete, pa i igraće konzole. Ova će želja biti još više naglašena u nasljedniku ovog operacijskog sustava. Razvijena je još jedna verzija IE 10, koja je bila prilagođena za korištenje dodirom. Upravljačka ploča je također dobila svoju verziju prilagođenu za dodir. Dodan je i izbornik Charmsbar, koji se uvijek nalazio na desnom rubu ekrana, ali se pojavljuje samo dodirom ruba pokazivačem miša. Iako je Windows 8 krenuo u pravom smjeru za Windows ekosustav, njegovo izvođenje je bilo loše zbog manjka aplikacija s Windows Store-a te uklanjanja Start izbornika, pa je zbog toga imao jako malu popularnost. Objavljen je u 3 verzije: Windows 8, Windows 8 Pro i Windows 8 Enterprise. Postojala je i verzija Windows Server 2012, a kasnije i njegovo R2 izdanje. Servisni paketi nisu objavljivani zbog toga što se sustav nadograđivao postepeno, stalnim instalacijama manjih ažuriranja. Višejezgreni procesori su u doba operacijskog sustava Windows 8 bili prisutni u većini računala, pa je softver na svim razinama bio prilagođen tako da koristi više jezgri, što je donijelo bolje performanse i efektivniju raspodjelu rada. Ukidanje Start izbornika je prekinulo tradiciju s prethodnim verzijama operacijskog sustava Windows, Start izbornik se nalazio u svakoj verziji Windows-a od 1995. godine, što je uz nezadovoljstvo korisnika natjeralo Microsoft da izda operacijski sustav Windows 8.1 kao nadgradnju na Windows 8. Operacijski sustav Windows 8.1 je objavljen 17. listopada 2013. godine kao besplatno preuzimanje za sve korisnike operacijskog sustava Windows 8. Start izbornik je donekle vraćen iako je Start gumb otvarao Start ekran umjesto Start izbornika, ali je koncept radne površine bio prisutan, što je smirilo veliku većinu korisnika. Ova verzija je dijelila identičan kod sa ažuriranom verzijom Windows-a 8, pa se zbog toga ne smatra samostalno razvijenim operacijskim sustavom. Također je u skladu sa imenom, objavljena i mobilna verzija Windows Phone 8.1 koja je donijela mnoštvo sitnih promjena. Podrška za Windows 8 je završila 12. siječnja 2016. godine, a za Windows 8.1 će završiti 10. siječnja 2023. godine
