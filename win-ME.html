Windows Millennium Edition

Windows ME

Windows ME (Millennium edition) je treći sustav Microsoft-ove Windows 9x obitelji sustava. Iako je prije njega objavljen sustav Windows 2000 sa Windows NT arhitekturom, Windows ME je izravni nasljednik sustava Windows 98, dok je Windows 2000 konačni rezultat višegodišnjeg eksperimenta (više o tome u sljedećem poglavlju). Dakle, Windows ME nastavlja s razvojem svojih prethodnika, objavljen je 14. rujna 2000. godine, te je nazivom simbolično proslavio ulaz čovječanstva u novo tisućljeće. Najveće promjene donose novi programi, a najznačajniji su Internet Explorer 5.5 (nadalje IE), WMP 7, potpuno novi Windows Movie Maker za stvaranje video uradaka, Vraćanje sustava, automatizirano ažuriranje zatim Windows DVD player, a za igre izuzetno važan DirectX 9.0c, kojeg podržava i većina suvremenih igara. Za Windows ME se s pravom može reći da je usavršeni Windows 98, te da je vrhunac i zadnji član obitelji Windows 9x. Opća podrška za Windows ME je završila krajem 2003. godine.
