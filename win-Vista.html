    Windows Vista

    Windows Vista

    Windows Vista ili projekt Longhorn se počeo razvijati već u svibnju 2001. godine, a objavljen je 30. siječnja 2007. godine. Ono po čemu je ovaj operacijski sustav najpoznatiji je njegova Aero prozirna tema, koja je s jedne strane donijela nevjerojatan vizualni doživljaj, ali je s druge strane zahtijevala izuzetno jak hardver za svoje izvođenje. Kombinacija visokih zahtjeva i mnogih sigurnosnih podsustava, kao što je Windows UAC (eng. User Account Control – kontrola korisničkog računa) dovela je do toga da Windows Vista nije uspjela zadobiti veliku popularnost u odnosu na već dobro udomaćeni Windows XP, koji postaje njenim rivalom. Nakon početnih kontroverzi, dokazalo se da upravljački programi za Windows XP ne rade pravilno na operacijskom sustavu Windows Vista, pa su mnoge tvrtke ostale na Windows-u XP. Ipak, Windows Vista je donijela mnoštvo unaprjeđenja sigurnosti kao npr. već spomenuti Windows UAC, koji je podigao sigurnost na višu razinu. Windows Vista je uz svoju prozirnu Aero temu imala još vizualnih dodataka u odnosu na Windows XP, a neki od njih su: Windows Sidebar i Windows Search (koji omogućava instant pretragu datoteka). Što se tiče Microsoft-ovih programa, WMP je podignut na verziju 11, inkorporirajući prozirnu Aero temu u svoje sučelje, a IE je dobio svoju sedmu iteraciju (verzija 7.0). Za ubrzavanje Windows-a Vista operacijskog sustava korištena je umjetna inteligencija, koja je upravljala izuzetno naprednim servisom Superpredmemoriranja (eng. Superfetch), ona ista tehnologija koja je ubrzala Windows XP sada je usavršena u potpunosti, i to kroz tehnologije ReadyBoost i ReadyDrive koje su ubrzale pokretanje i korištenje sustava, a na prijenosnim računalima su produžile i vijek trajanja baterije. Windows Vista je objavljena u šest verzija: Windows Vista Starter, Windows Vista Home Basic, Windows Vista Home Premium, Windows Vista Business, Windows Vista Enterprise i Windows Vista Ultimate. Za Windows Vista operacijski sustav objavljena su dva servisna paketa i jedna nadgradnja platforme, koja je donijela neke komponente iz nasljednika ovog sustava, Windows-a 7. Windows Vista je svojim brojnim mogućnostima i prelijepim vizualnim doživljajem postala baza za razvoj svog nasljednika, te se većina stvari za Windows 7 preuzela direktno iz ovog sustava. Postojala je i serverska verzija ovog sustava, pod nazivom Windows Server 2008. Podrška za operacijski sustav Windows Vista završila je 11. travnja 2017. godine.
